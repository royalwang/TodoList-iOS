Cloud Foundry

Build, test, and deploy apps on Bluemix by using open source Cloud Foundry technology. Our Platform as a Service gets you up and running, with no need to manage infrastructure.

View Docs View Labs

Get started quickly with Cloud Foundry:

  1. Download and install the Cloud Foundry command line interface.
  2. Create an app from a starter in the Bluemix Catalog or click CREATE AN APP to get a guided experience.
  3. On the app Overview page, click Start Coding to download the starter code, make changes, and run your modified app.

IBM Containers

Use IBM Containers to run apps and services in a hosted cloud environment. Port your existing applications to IBM Bluemix and make them publicly accessible and composable. Use a private registry to upload, store, and retrieve your trusted images.

View Docs View Labs

Get started quickly with IBM Containers:

  1. Download and install Docker, the Cloud Foundry Command Line interface, and the IBM Containers plug-in for Cloud Foundry.
  2. Log in to Bluemix with your IBM id and password: 
    cf login
  3. Log in to the IBM Containers service and private registry with your IBM id and password: 
    cf ic login
  4. In the root directory of your app source files, create a Dockerfile.
  5. From the directory with the Dockerfile, build a named Docker image: 
    docker build -t <image_name> .
  6. Tag the image with your private registry: 
    docker tag <image_name> registry.{DOMAIN}/{namespace}/<image_name>:<image_tag>
    The <image_tag> is optional; if omitted, it defaults to latest.
  7. Push the image to the private registry for your organization: 
    docker push registry.{DOMAIN}/{namespace}/<image_name>:<image_tag>
  8. Create and start a container from the image in the Bluemix catalog or by using the following command: 
    cf ic run --name <container_name> registry.{DOMAIN}/{namespace}/<image_name>

Virtual Servers Beta

Get total control over your app's infrastructure by deploying virtual servers. Manage them from the Bluemix Dashboard, or from anywhere with OpenStack APIs.

View Docs

Get started quickly with virtual servers:

  1. Click CREATE A VIRTUAL SERVER from the Bluemix Dashboard.
  2. Select one of the available virtual server images.
  3. Optionally, customize the name or size.
  4. Click Next and finish creating the virtual server.
  5. Start the Virtual Server.
